# 🔒 Security Policy

## Xavfsizlik muammolarini xabar berish

Agar loyihada xavfsizlik muammosi topsangiz, iltimos **ommaviy issue ochmang**.

Email: **security@00o.uz**  
Telegram: **@mira_support_team** (private message)

24 soat ichida javob beramiz.

## Qo'llab-quvvatlanadigan versiyalar

| Versiya | Qo'llab-quvvatlanadi |
|---------|---------------------|
| 1.x     | ✅ Aktiv             |
| < 1.0   | ❌ Yo'q              |

## Xavfsizlik choralari

- ✅ JWT with refresh tokens
- ✅ bcrypt password hashing
- ✅ Helmet (security headers)
- ✅ CORS configured
- ✅ Rate limiting
- ✅ Zod input validation
- ✅ Prisma (SQL injection safe)
- ✅ XSS protection
- ✅ CSRF tokens
- ✅ Environment variables
- ✅ Telegram HMAC verification
- ✅ HTTPS only (production)

## Hisob ma'lumotlari

Agar akkauntingiz buzilgan deb hisoblasangiz:
1. Parolni darhol o'zgartiring
2. 2FA yoqing (agar mavjud bo'lsa)
3. Support'ga murojaat qiling

Rahmat! 🛡
